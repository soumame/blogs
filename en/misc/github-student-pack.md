# What is the GitHub Student Pack?

The GitHub Student Pack is the name of a package through which GitHub provides paid software, free server credits, the GitHub Pro plan, and other benefits to students at no cost.

In Japan, you can use it simply by registering a photo of your student ID or a school-issued email address. I don't know the total monetary value, but there's so much included that you can't possibly use it all, so anyone who uses GitHub should definitely sign up. It's one of those things where you'd be losing out if you don't.

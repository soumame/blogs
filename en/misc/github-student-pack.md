# What is the GitHub Student Pack?

The GitHub Student Pack is the name of a package through which GitHub provides students with paid software, free server credits, the GitHub Pro plan, and other benefits at no cost.

In Japan, you can access it simply by registering a photo of your student ID or a school-issued email address. I don’t know the total monetary value, but there are so many perks that you’ll probably never use them all, so anyone who uses GitHub should definitely sign up. Not signing up is essentially leaving money on the table.

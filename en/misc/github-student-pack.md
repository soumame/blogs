# What is the GitHub Student Pack?

The GitHub Student Pack is a package through which GitHub provides students with paid software, free server credits, GitHub Pro plans, and more at no cost.

In Japan, you can get it simply by registering a photo of your student ID or a school-issued email address. I don't know the exact total value, but it's so generous that you probably won't be able to use everything, so anyone who uses GitHub should definitely sign up. You'd be missing out if you don't.

# What is the GitHub Student Pack?

The GitHub Student Pack is a bundle through which GitHub provides students with paid software, free server credits, GitHub Pro plans, and other perks at no cost.

In Japan, you can apply simply by registering a photo of your student ID or a school-issued email address. I don't know the total monetary value, but there are so many benefits that you probably won't be able to use them all, so anyone who uses GitHub should definitely sign up. It's the kind of deal where you lose out if you don't.
